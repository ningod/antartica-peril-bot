# ANTARTICA — Translation Glossary

_Italian → English — Game Terminology Reference_

Each entry includes the original Italian term, the English translation, and a brief description of its role within the game. Notes on plurals and gender are provided where relevant.

---

## Tags & Character Elements

| Italiano           | English          | Description                                                                                                                                                                                                                                            |
| ------------------ | ---------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| **Etichetta**      | **Tag**          | The physical token (label) representing any game element. Players write keywords on Tags and place them in the Pouch. Tags are the core physical component of all game mechanics.                                                                      |
| **Tratto**         | **Trait**        | A Tag representing a positive character aspect: personality, skills, physical or psychological qualities. Never an object. Traits are positive when drawn. _Plural: Traits._                                                                           |
| **Tratto-Segnato** | **Marked Trait** | A Trait that has degenerated into its negative counterpart, written on the reverse side of the Tag. When drawn, if the negative side comes up, it counts as a negative result. E.g. Intelligent / Arrogant; Tenacious / Stubborn.                      |
| **Archetipo**      | **Archetype**    | A Trait subtype indicating the character's role aboard the _Endurance_. Defines the character's function within the expedition crew.                                                                                                                   |
| **Nome**           | **Name**         | A Trait subtype: the character's proper name.                                                                                                                                                                                                          |
| **Risorsa**        | **Supply**       | A Tag representing a physical object or equipment: ropes, grappling hooks, rifles, provisions. Supplies are positive when drawn and can be Spent as a consequence. _Plural: Supplies._                                                                 |
| **Minaccia**       | **Threat**       | A story Tag representing a foreseeable adverse event or moment. Threats are the Stakes of each Scene and are Checked off when confronted. Negative when drawn. _Plural: Threats._                                                                      |
| **Visione**        | **Vision**       | A Threat subtype: unexpected, often supernatural events. Negative when drawn. _Plural: Visions._                                                                                                                                                       |
| **Condizione**     | **Condition**    | A Tag representing physical or psychological damage, wounds, or trauma afflicting a character. Negative when drawn. _Plural: Conditions._                                                                                                              |
| **Terrore**        | **Dread**        | A Condition subtype: primarily psychological harm, an obsession or phobia. Negative when drawn. _Note: "Dread" is preferred over literal "Terror" to convey lingering, creeping fear rather than sudden shock, fitting the Antarctic isolation theme._ |
| **Rassegnazione**  | **Resignation**  | A special unwritten Tag representing a character's immobility and surrender. Negative when drawn.                                                                                                                                                      |

## Core Mechanics

| Italiano                   | English                  | Description                                                                                                                                                                                                                                                                                                                                                                             |
| -------------------------- | ------------------------ | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Guida**                  | **Lead**                 | The character who takes the primary role in a Brave the Peril procedure. The Lead contributes the most Tags to the Pouch and acts as the narrator of the action. Other characters must have Bonds with the Lead in order to Aid effectively. _Note: "Lead" is preferred over literal "Guide" to avoid confusion with the GM or rulebook, common meanings of "guide" in English TTRPGs._ |
| **Collaborare**            | **Aid**                  | To add positive Tags to the Pouch when not acting as the Lead, helping achieve the same objective. Requires a Bond with the Lead. _Note: as a verb, "to Aid"; as a noun for the action itself, "Aid" is also valid (e.g. "providing Aid")._                                                                                                                                             |
| **Sacchetto**              | **Pouch**                | The physical container (bag) into which Tags are placed before a Draw. Replaces dice as the core randomization method.                                                                                                                                                                                                                                                                  |
| **Estrazione**             | **Draw**                 | The act of pulling Tags from the Pouch to determine outcomes. The standard Draw pulls 3 Tags, with an optional Push for 2 more.                                                                                                                                                                                                                                                         |
| **Affrontare il Pericolo** | **Brave the Peril**      | The core resolution procedure. Characters face danger by Drawing Tags from the Pouch. Positive Tags (Traits, Supplies) mean success; negative Tags (Conditions, Dread, Threats, Visions, Resignation, Marked Traits) mean complications.                                                                                                                                                |
| **Superarsi**              | **Push**                 | The optional second phase of Brave the Peril: drawing 2 additional Tags, risking worse consequences for a chance at Growth. Echoes the "push your luck" mechanic common in modern TTRPGs.                                                                                                                                                                                               |
| **Evoluzione**             | **Growth**               | The positive outcome of a successful Push: the character may gain a new Trait or remove the negative side of a Marked Trait.                                                                                                                                                                                                                                                            |
| **Conseguenze Nefaste**    | **Dire Consequences**    | The negative outcome of a failed Push: the character must Spend a Supply or convert a Trait into a Marked Trait.                                                                                                                                                                                                                                                                        |
| **Consumare**              | **Spend**                | To lose, break, or use up a Supply Tag as a consequence of failure. The Tag is removed from play.                                                                                                                                                                                                                                                                                       |
| **Difficoltà**             | **Challenge**            | An obstacle or danger within a Scene that characters must overcome, typically by Braving the Peril. _Plural: Challenges._                                                                                                                                                                                                                                                               |
| **Posta in Gioco**         | **Stakes**               | What is at risk in each Scene. Narratively, it is advancing Shackleton's Plan; mechanically, it means Checking off a Threat or Vision Tag.                                                                                                                                                                                                                                              |
| **Spuntare**               | **Check**                | To mark a Threat or Vision Tag as confronted, checking it off. Tied to the outcome of Brave the Peril.                                                                                                                                                                                                                                                                                  |
| **Diario di Shackleton**   | **Shackleton's Journal** | The game's narrative guide, structuring the expedition story and providing historical context.                                                                                                                                                                                                                                                                                          |
| **Relazione**              | **Bond**                 | An interpersonal connection formed between characters during Interludes. _Plural: Bonds._                                                                                                                                                                                                                                                                                               |
| **Successivo**             | **Next**                 | Indicates progression to the following Act, Scene, or game phase.                                                                                                                                                                                                                                                                                                                       |

## Game Structure

| Italiano       | English       | Description                                                                                                                    |
| -------------- | ------------- | ------------------------------------------------------------------------------------------------------------------------------ |
| **Atto**       | **Act**       | A major division of the game's structure, like an act in a play. The game is divided into sequential Acts. _Plural: Acts._     |
| **Scena**      | **Scene**     | A segment within an Act where characters face Challenges and Brave the Peril. Each Scene has its own Stakes. _Plural: Scenes._ |
| **Intermezzo** | **Interlude** | A segment within an Act for rest and forming Bonds between characters. A quieter counterpart to Scenes. _Plural: Interludes._  |

---

## Quick Reference: Draw Outcomes

**Positive (success):** Trait, Supply, Name, Archetype

**Negative (complication):** Condition, Dread, Threat, Vision, Resignation, Marked Trait (negative side)
